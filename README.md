# expense-tracker-python
Expense management system built with Python featuring expense tracking, category-wise analysis, and persistent JSON-based storage.
## Features
- Add expenses
- View expense history
- Category-wise spending summary
- Total spending calculation
- JSON-based data storage

## Technologies Used
- Python
- JSON
- File Handling

## Live Demo
https://expense-data-loader--shauryavivek090.replit.app

## Project Structure

main.py - Main application code
expenses.json - Stores expense records

## How to Run

1. Clone the repository
2. Run:
```bash
python main.py
```
